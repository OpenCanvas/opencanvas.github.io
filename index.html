Welcome to OpenCanvas open source portal!

In case you are here by accident, we remind you that OpenCanvas is an innovative platform for managing rich content and creating beautiful location based mobile apps with it.

Here we host a collection of open source projects that may aid you, in case you choose to create your own apps using the OpenCanvas API, instead of using the web version of the platform.

Projects

For now, we only showcase a collection of sample applications that demonstrate the usage of the OpenCanvas API. You can find the sample projects in the respective repositories for iOS, Android and Javascript.
